## 2. Новые геймплейные механики

| Механика | Описание | Сложность |
|----------|----------|-----------|
| **Шкала Суперкомбо (EX Meter)** | Полоска EX: +5% за нанесённый урон, +10% за полученный/блокированный. При 100% — бесплатная "Ультра-Комбо" с огромным уроном. | Средняя (новая переменная + UI-полоска) |
| **Чистка колоды (Deck Thinning)** | На экране победы: опция УДАЛИТЬ или УЛУЧШИТ базовую карту из колоды. | Низкая (UI + логика фильтрации deck) |
| **Guard Crush (стамина блока)** | Шкала выносливости защиты. Блоки её расходуют. При 0 — оглушение + защита сброшена. | Средняя (новая переменная + переработка блока) |
| **Стойки (Stances)** | Бесплатно 1 раз за ход: Атакующая стойка (+2 урон, нельзя блок) / Защитная (+10 HP барьер, урон снижен). | Средняя (новая UI + логика стоек) |

**Рекомендация:** начать с "Чистки колоды" — легко, большой геймплейный эффект.

## 3. Найденные баги и проблемы баланса

| Приоритет | Проблема | Детали | Фикс |
|-----------|----------|--------|------|
| 🔴 Высокий | **Эксплойт бесконечного стана** | `ice_spear` (2 хода заморозки) + Дхалсим (-1 эн на карты). Нет диминишинга контроля. | Добавить diminishing returns: каждая заморозка -1 ход (2→1→0) |
| 🟡 Средний | **"Удар в прыжке" бесполезен** | 2 эн, 18 урона, 50% точность. EV = 9 урона за 2 эн. `m_atk` даёт 18.5 за 2 эн. | Увеличить урон до 28 или точность до 75% |
| 🟡 Средний | **AI слепой к баффам игрока** | AI не проверяет dodgeChance/blockPct игрока перед атакой. Тратит спецприёмы "в молоко". | Добавить в `evalEnemyCard` проверку баффов противника |
| 🟡 Средний | **Баг комбо с многихударными картами** | `nextAtkBoost` обнуляется на `i === hits-1`, а не после последнего хита. `double_atk` теряет бафф. | Перенести сброс nextAtkBoost после цикла hits |
| 🟡 Средний | **Вампиризм масштабируется с баффами** | `v_atk` лечит на 50% от *фактического* урона. С комбо/Зангиевом → 20-30 HP за удар. | Сделать вампиризм фиксированным (5-8 HP) или от базового урона карты |
| 🟢 Низкий | **"Удар в прыжке" нестабилен** | 50% шанс промаха без контроля — фрустрирует игрока. | Убрать шанс, добавить trade-off (например, игнор блока + 50% урона) |

## 4. Оценка бэклога

| Фича | Реалистичность | Оценка | Примечание |
|------|----------------|--------|------------|
| **Мультиплеер hotseat** | ✅ Очень реалистично | ~50-70 строк | Ширма при передаче хода, переключение activePlayer |
| **Анимации карт** | ✅ Очень реалистично | CSS keyframes | Раздача из колоды, растворение при сбросе |
| **Звуковые эффекты** | ✅ Реалистично | Расширить Audio.playTone | Noise buffer для взрывов, блока |
| **Таблица лидеров (глобальная)** | ❌ Нереалистично | Требует backend | В рамках 1 HTML-файла невозможно. Локальные рекорды — да. |
